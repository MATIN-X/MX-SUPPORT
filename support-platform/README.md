# Support Platform

A complete support platform with Telegram authentication, PWA functionality, and responsive design.

## Features

- Telegram authentication for users
- Guest user support
- Admin panel at `/Support`
- User interface at `/User`
- SSL certificate setup
- PWA functionality
- Responsive design
- Database integration

## Installation

Run the installation script:

```
chmod +x install.sh
./install.sh
```

## Usage

After installation, access:
- User interface: `https://yourdomain.com/User`
- Admin panel: `https://yourdomain.com/Support`
